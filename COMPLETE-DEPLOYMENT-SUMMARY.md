## üí¨ FAQ ABOUT YOUR NEW SITE

**Q: Why aren't all 9 photos visible on the homepage?**
A: 4 are on homepage (headshot + 3 gallery). The other 5 are ready for Skills page, Work page, and blog posts.

**Q: Will the headshot display this time?**
A: Yes! The file is named correctly and in the right location. If it doesn't show, it's likely a browser cache issue - try incognito mode.

**Q: Can I add more photos later?**
A: Absolutely! Just upload them to GitHub and reference them in the HTML.

**Q: How do I update the blog categories?**
A: Each blog card has `data-category="professional"` or `data-category="personal"`. Change this attribute to change the category.

**Q: How do I add more FAQs?**
A: Copy an existing FAQ item div and modify the question/answer text.

**Q: Can I change the colors?**
A: Yes! All colors are defined in CSS variables at the top of the stylesheet. Change the hex codes to update the theme.
